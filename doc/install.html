Install

  • - Lancez le script /src/bd/bd.sql dans votre gestionnaire MySQL (phpmyadmin, sequel pro ...) pour créer la base de donnée
  • - Modifiez les valeurs de connexion a la base de données dans le fichier /src/web/env.php
  • - Glissez le dossier /src/web dans votre dossier racine de votre serveur ou créez un host vers ce dossier
